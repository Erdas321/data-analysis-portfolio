How to Install and Open Power BI

1. Download Power BI
You can download Power BI Desktop for free from the official Microsoft website:
ðŸ”— Power BI Download

2. Install Power BI
Open the downloaded file and follow the installation steps.

You may need to sign in with a Microsoft account to use Power BI.

3. Open the Power BI Project
After installing, open Power BI Desktop.

Click File > Open and select the .pbix file from this repository.

If prompted, allow data refresh to load the latest data.

4. Explore the Dashboard
Navigate through different pages of the dashboard.

Use filters and slicers to interact with the data.
