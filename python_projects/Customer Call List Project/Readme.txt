Customer Call List
Project Overview
The Customer Call List project utilizes a Jupyter Notebook to process and analyze customer data stored in an Excel file. It helps manage customer interactions by organizing contact information and call schedules.​

Project Files
Customer_Call_List.ipynb: Jupyter Notebook containing code to load, process, and analyze customer data.​

customer_data.xlsx: Excel file with customer information, including names, contact details, and call history.​

Requirements
Python Version: Python 3.8 or higher.​

Jupyter Notebook: Interactive computing environment for running the notebook.​

Libraries: Standard Python libraries such as pandas for data manipulation and openpyxl for reading Excel files.​

​Note: These requirements are typically met if you have Anaconda installed.​

Usage
Launch Jupyter Notebook:

Open Anaconda Navigator.​
Medium

Select your desired environment (e.g., "base (root)").​

Click "Launch" under Jupyter Notebook.​

Open and Run the Notebook:

In the Jupyter Notebook interface, navigate to the directory containing Customer_Call_List.ipynb and customer_data.xlsx.​

Open Customer_Call_List.ipynb.​

Change path for customer_data.xlsx as needed

Execute the cells sequentially to process and analyze the customer data.
